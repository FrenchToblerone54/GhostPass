# GhostPass

**GhostPass** یک ربات تلگرام پایتون با مجوز MIT است که به عنوان فروشگاه VPN برای کاربران نهایی، روی یک نمونه [GhostGate](https://github.com/FrenchToblerone54/ghostgate) عمل می‌کند.

**معماری:** کاربر → ربات GhostPass → GhostGate REST API → نودهای 3x-ui

---

## شروع سریع

```bash
wget https://raw.githubusercontent.com/FrenchToblerone54/ghostpass/main/install.sh -O install.sh
chmod +x install.sh
sudo ./install.sh
```

اسکریپت چهار چیز می‌پرسد:
1. **توکن ربات** — از [@BotFather](https://t.me/BotFather)
2. **شناسه عددی تلگرام مدیر** — آیدی عددی تلگرام شما
3. **زبان** — `en` (انگلیسی) یا `fa` (فارسی)
4. **آدرس پروکسی ربات** — اختیاری، اگر نیاز نیست خالی بگذارید

همه چیز دیگر از طریق ویزارد راه‌اندازی اولیه ربات تنظیم می‌شود.

---

## ویزارد راه‌اندازی اولیه

پس از نصب، تلگرام را باز کنید و به عنوان مدیر `/start` را به ربات ارسال کنید. ویزارد شما را راهنمایی می‌کند:

1. **آدرس GhostGate** — آدرس کامل پنل شامل مسیر مخفی (مثال: `https://vpn.example.com/mySecretPath`). ربات قبل از ذخیره اتصال را تست می‌کند.
2. **تماس پشتیبانی** — یک @username که کاربران برای پشتیبانی به آن مراجعه کنند.
3. **پرداخت کارت به کارت** — شماره کارت و نام دارنده کارت (روش اصلی برای کاربران ایرانی).
4. **Cryptomus** — Merchant ID و API Key (اختیاری، برای پرداخت رمزارزی).
5. **واحد ارز** — مثلاً `IRR`، `USD`، `USDT`.

## پنل مدیریت

مدیران از طریق `/start` به رابط مدیریت دسترسی دارند. منو کاملاً با کیبورد اینلاین کار می‌کند و در منوی کاربران نمایش داده نمی‌شود.

### اشتراک‌ها
- مشاهده همه اشتراک‌ها به‌صورت زنده از GhostGate (صفحه‌بندی شده، قابل جستجو)
- مشاهده آمار و QR کد هر اشتراک
- ایجاد دستی اشتراک با پارامترهای دلخواه و انتخاب نود
- حذف اشتراک

### پلان‌ها
- ایجاد، ویرایش و حذف پلان
- انتخاب نودها/اینباندهای GhostGate برای هر پلان
- فعال/غیرفعال کردن
- ویرایش قیمت و نام به‌صورت اینلاین

### کاربران
- جستجو با شناسه تلگرام یا @username
- مشاهده اطلاعات کاربر و تاریخچه سفارشات
- مسدود/رفع مسدودیت کاربران

### مدیران
- افزودن مدیران جدید با شناسه تلگرام و مجوزهای قابل تنظیم
- حذف مدیران (مدیر اصلی قابل حذف نیست)

### سفارشات
- مشاهده سفارشات در انتظار تأیید
- مشاهده تاریخچه سفارشات پرداخت‌شده و رد‌شده
- تأیید یا رد دستی پرداخت‌های کارتی

### تنظیمات
- **اتصال GhostGate** — به‌روزرسانی URL با تست اتصال زنده
- **کارت به کارت** — فعال/غیرفعال، ویرایش شماره کارت و نام دارنده
- **Cryptomus** — فعال/غیرفعال، ویرایش Merchant ID و API Key
- **جریان درخواست** — فعال/غیرفعال امکان درخواست اشتراک
- **تماس پشتیبانی** — به‌روزرسانی @username پشتیبانی
- **واحد ارز** — به‌روزرسانی ارز نمایشی
- **فاصله زمانی همگام‌سازی** — به‌روزرسانی فاصله همگام‌سازی پس‌زمینه

## روش‌های پرداخت

### کارت به کارت (اصلی)
بدون نیاز به هیچ تنظیم دیگری کار می‌کند. کاربران:
1. پلانی را انتخاب می‌کنند → اطلاعات کارت را می‌بینند
2. تصویر رسید را ارسال می‌کنند
3. مدیر رسید را با دکمه‌های تأیید/رد دریافت می‌کند
4. پس از تأیید، اشتراک به‌صورت خودکار ایجاد می‌شود

### Cryptomus (اختیاری)
از طریق تنظیمات → Cryptomus پیکربندی کنید. فقط در صورت تنظیم نمایش داده می‌شود.

آدرس وب‌هوک برای تأیید سریع‌تر پرداخت:
```
http://YOUR_SERVER:8090/webhook/cryptomus
```

ثبت‌نام در [cryptomus.com](https://cryptomus.com).

### جریان درخواست (اختیاری)
از طریق تنظیمات → جریان درخواست فعال کنید. کاربران درخواستی با دلیل اختیاری ارسال می‌کنند؛ مدیر تأیید یا رد می‌کند.

## راهنمای متغیرهای .env

| متغیر | تنظیم توسط | توضیح |
|---|---|---|
| `BOT_TOKEN` | install.sh | توکن ربات تلگرام از @BotFather |
| `ADMIN_ID` | install.sh | شناسه عددی تلگرام مدیر اصلی |
| `BOT_PROXY` | install.sh | آدرس پروکسی اختیاری برای API تلگرام |
| `LANGUAGE` | install.sh | `en` یا `fa` |
| `GHOSTGATE_URL` | ویزارد راه‌اندازی | آدرس کامل پنل GhostGate با مسیر مخفی |
| `SYNC_INTERVAL` | منوی تنظیمات | فاصله همگام‌سازی پس‌زمینه بر حسب ثانیه |
| `DB_PATH` | install.sh | مسیر پایگاه داده SQLite |
| `LOG_FILE` | install.sh | مسیر فایل لاگ |

اطلاعات پرداخت (شماره کارت، کلیدهای Cryptomus) در جدول `settings` پایگاه داده SQLite ذخیره می‌شوند و هرگز در `.env` قرار نمی‌گیرند.
